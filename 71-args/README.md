# Aula 71 - *args Para quantidade de Argumentos não nomeados

Nesta aula, compreendi o uso do `*args` para funções que precisam receber uma quantidade variável de argumentos.



### ✅ Itens para o seu Checklist:
* [x] Criar funções que aceitam múltiplos argumentos com *args.
* [x] Utilizar o laço for para processar os elementos de uma tupla args.
* [x] Desempacotamento: Usar o * para enviar os itens de uma lista/tupla individualmente para uma função.
